## Button 按钮

  <button-demo/>
<font size=5>Attributes</font>
| 参数| 说明 | 类型 | 可选值 | 默认值 |
| :------ | ------ | ------ | ------ | ------ |
| type | 按钮类型 | String | success, primary, danger, warning, default | default |

| loading | 加载状态 | Boolean | false | - |
| size | 按钮尺寸 | String, Number | 1, 2, 3 | 2 |
